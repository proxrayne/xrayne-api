# Shared .NET Backend Styleguide

The canonical Proxrayne .NET backend styleguide now lives in the meta
repository:

```text
../docs/DOTNET_STYLEGUIDE.md
```

Use that shared guide for backend changes in this repository, together with the
panel-specific docs:

- `docs/architecture/backend.md`
- `docs/conventions/api.md`
- `docs/project-rules.md`
- `docs/styleguide/dotnet.md`

Panel-specific docs may add local constraints, but they must not weaken the
shared styleguide.
